easy galaxy(jarvisoj)

打开网页，可以看见一个上传图片的页面，首先编写一个php脚本，改变后缀名为jpg，然后上传，但是显示只能上传jpg文件或者gif文件，怀疑是基于MIME
来进行比较的，也就是根据text/html这种类型的格式去判断的，这种类型不能随便修改。这样子只能上传一张真正的图片了，但是可以在上传的图片结尾加上图片 木马，如图(好像崩了)：

上传之后，回显得到图片的id，多猜一猜可以发现是在
http://web.jarvisoj.com:32785/uploads/1499355842.jpg下面，然后再看看其他页面的url
http://web.jarvisoj.com:32785/index.php?page=view尝试构造http://web.jarvisoj.com:32785/index.php?page=uploads/1499355842.jpg但是，页面显示说1499355842.jpg.php找不到，可以考虑%00来截断，尝试构造http://web.jarvisoj.com:32785/index.php?page=uploads/1499355842.jpg%00,然后就
出来了flag，但是如果是上传一个没有getshell的图片，他是不会弹出来flag的，大概他是去检测匹配了文件内容了吧，如果有getshell就出flag
